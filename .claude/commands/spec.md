구글 드라이브에서 와썹하우스 명세서를 읽고, "$ARGUMENTS"와 관련된 내용만 찾아서 요약한다.

다음 순서로 진행한다:

1. Google Drive MCP를 사용해 와썹하우스 관련 문서를 검색한다.
   - 요구사항 명세서
   - API 명세서 (있다면)
   - 화면 설계서 또는 UI 명세서 (있다면)

2. "$ARGUMENTS"와 직접 관련된 섹션만 골라낸다. 전체 문서를 복사하지 않는다.

3. 아래 형식으로 요약한다:

---

## 기능 요약
(어떤 기능인지 2-3문장)

## 요구사항
(구현해야 할 항목을 bullet로)

## UI / 화면 흐름
(있다면: 어떤 화면이 필요한지, 어떤 순서로 동작하는지)

## API 연동
(있다면: 어떤 엔드포인트를 호출하는지, 요청/응답 구조)

## 제약 조건
(있다면: 권한, 예외 케이스, 비즈니스 규칙)

---

관련 명세가 없다면 "명세서에서 관련 내용을 찾지 못했습니다"라고 알린다.
